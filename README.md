# 🛡️ TgPcAgent

**Удалённое управление ПК через Telegram-бота с невероятно быстрой облачной архитектурой.**

[![Latest Release](https://img.shields.io/github/v/release/ExtraTools/TgPcAgent?label=Releases&color=blue)](https://github.com/ExtraTools/TgPcAgent/releases/latest)
[![Bot Link](https://img.shields.io/badge/Telegram-Bot-blue?logo=telegram)](https://t.me/WaitDino_bot)

---

## ✨ Что умеет агент?

* 📸 **Снимки экрана:** Быстрая отправка в формате JPEG.
* 📊 **Мониторинг:** Температура GPU Core, загрузка CPU и использование RAM в реальном времени.
* ⚙️ **Процессы:** Просмотр и принудительное завершение программ (`TaskManager`).
* 🚀 **Запуск приложений:** Каталог ваших программ с быстрым запуском.
* 🔌 **Управление питанием:** Выключение, перезагрузка, сон и блокировка (с защитой от случайных нажатий).
* 🔄 **Автоматизация:** Задание фона (авто-пинг, создание скриншотов по расписанию).
* ☁️ **Cloud Relay:** Безопасная мультипользовательская архитектура. Агент не требует белого IP или проброса портов.
* 📦 **Silent Updates:** Апдейтер сам скачивает новые релизы с GitHub и обновляется в фоне.

## 🛠 Как установить?

1. Перейдите в раздел **[Releases](https://github.com/ExtraTools/TgPcAgent/releases/latest)** и скачайте свежий `TgPcAgent-Setup.exe`.
2. Запустите установщик.
3. Дождитесь появления иконки агента в трее (возле часов на панели задач Windows).
4. Кликните по иконке правой кнопкой мыши и нажмите **«Показать код привязки»**.
5. Зайдите в официального бота **[@WaitDino_bot](https://t.me/WaitDino_bot)** и отправьте: `/pair ВАШ_КОД`.

## 🏗 Архитектура

Проект построен на асинхронном пуле команд через Vercel. 
ПК работает только на pull-запросы (в целях безопасности порты не открываются).
