## ✨ Что умеет агент?

* 📸 **Снимки экрана:** Быстрая отправка свежих скриншотов рабочего стола (`/screenshot`).
* 📊 **Мониторинг:** Температура GPU/CPU, загрузка процессора, использование RAM, время аптайма и локальные IP-адреса (`/status`).
* ⚙️ **Процессы:** Просмотр топа процессов по памяти и возможность их завершения (`/processes`).
* 🚀 **Запуск приложений:** Ваш личный каталог программ с быстрым запуском по алиасу (`/open`) и авто-поиск ярлыков меню Пуск (`/scanapps`).
* 🔌 **Управление питанием:** Выключение, перезагрузка, сон и блокировка (выключение/перезагрузка защищены двойным подтверждением от случайных нажатий).
* 📜 **Скрипты и автоматизация:** Мощный движок для выполнения кастомных сценариев (`/script`), фоновые задачи (`/auto`) и встроенный файловый менеджер.
* ☁️ **Cloud Relay:** Инновационная мультипользовательская архитектура. Не требует «белого» IP или открытых портов (Telegram Webhook обрабатывается в облаке, агент делает pull-запросы).
* 🔐 **Приватность:** Авторизация ПК работает через уникальный код привязки (`/pair`). Доступ имеет только ваш аккаунт, остальные чаты игнорируются на уровне агента.
* 📦 **Silent Updates:** Апдейтер сам регулярно проверяет новые релизы на GitHub и автоматически обновляется в фоне.

## 🛠 Как установить и привязать?

1. Перейдите в раздел **[Releases](https://github.com/ExtraTools/TgPcAgent/releases/latest)** и скачайте свежий `TgPcAgent-Setup.exe` (Full или Lite версию).
2. Запустите установщик.
3. Дождитесь появления иконки агента в системном трее Windows (возле часов).
4. Кликните по иконке агента правой кнопкой мыши и нажмите **«Показать код привязки»**.
5. Зайдите в официального бота **[@WaitDino_bot](https://t.me/WaitDino_bot)** и отправьте команду: `/pair ВАШ_КОД`.

## 🏗 Архитектура

Проект построен на механизме асинхронной очереди команд через бессерверное облако Vercel с использованием Redis в качестве быстрого хранилища.
Агент работает исключительно на локальные pull-запросы — ваш ПК не "светит" портами наружу, что гарантирует максимальную безопасность.

```text
Ваш телефон (Telegram) → Bot Webhook (Vercel + Redis Queue) 
                                       ↓
                        (каждые 15s) Pull-запрос
                                       ↓
                                Ваша система (TgPcAgent)
```

Вся история команд, конфигурация и логи агента хранятся у вас локально в `%LocalAppData%\TgPcAgent\`.
