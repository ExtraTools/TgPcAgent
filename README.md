## 🚀 Что это такое

**TgPcAgent** — это система удалённого управления компьютером через Telegram.

Ты пишешь боту → команда улетает в облако → агент на ПК выполняет → результат возвращается обратно.

Работает даже если ПК за NAT, без проброса портов, без ебли с сетями.

---

## ⚡ Основные возможности

### 🖥 Управление ПК
- Скриншот экрана
- Просмотр процессов
- Запуск и убийство приложений
- Блокировка, перезагрузка, выключение
- Эмуляция клавиатуры и ввода текста

### 📊 Мониторинг
- CPU / RAM / диски
- Температуры (WMI)
- Аптайм системы
- Сетевые данные

### 📁 Работа с файлами
- Просмотр файлов и папок
- Чтение / запись
- Копирование / перемещение
- ZIP / UNZIP
- Отправка файлов в Telegram

### 🤖 Скрипты (очень мощная хрень)
- JSON-скрипты для автоматизации
- 29 встроенных инструментов
- Возможность запускать цепочки действий
- Генерация скриптов через AI (`/scripthelp`)

### 🌐 Сетевые фичи
- Работает через облако (без портов)
- Direct Mode (прямое подключение через туннель)
- Автоматический fallback

### 🔄 Автоматизация
- Авто-пинг
- Авто-скриншоты
- Фоновые задачи

### 🧠 Удобство
- Один бот → много пользователей → много ПК
- Переключение между ПК
- Tray-приложение (тихо сидит в фоне)
- Автообновления
